La prueba de chi cuadrado es una prueba estadÃ­stica sencilla que puede ser usada cuando se tienen datos de recuento de ocurrencias de categorÃ­as. El anÃ¡lisis puede ser definido como una:

  • Prueba de diferencia de conteos: para identificar si los conteos observados en las categorÃ­as de una variable son estadÃ­sticamente diferentes entre sÃ­.

  • Prueba de diferencia de conteos con probabilidades especÃ­ficas: identifica si los conteos observados en las categorÃ­as de una variable son estadÃ­sticamente diferentes entre sÃ­ dado que cada evento tiene una probabilidad de ocurrencia definida. Se debe especificar las probabilidades como un vector en el argumento p de la funciÃ³n. Para fines de este tutorial no veremos el detalle de esta prueba.

  • Prueba de independencia de dos factores: identifica a nivel estadÃ­stico si las categorÃ­as de dos variables estÃ¡n asociadas o son independientes.

En R, la funciÃ³n nativa bÃ¡sica para el cÃ¡lculo de esta prueba es:

# la correcciÃ³n de Yates activa por defecto.

Chi cuadrado para conteos

Se desea evaluar si existe diferencia entre la cantidad (recuento) de las personas inscritas en todos los grados para el primer nivel de la tabla grados podemos hacer un test de chi cuadrado sencillo:

# Separar la base de datos de grados
inscritos <- grados[1, ]
print(inscritos)
## Grado1 Grado2 Grado3 Grado4 Grado5 
##   2900    392    932   1812   2854
# Realizar la prueba chi cuadrado
resX2 <- chisq.test(inscritos, correct = F)

# Inspeccionar los valores esperados
print(resX2$expected)
## Grado1 Grado2 Grado3 Grado4 Grado5 
##   1778   1778   1778   1778   1778
# Inspeccionar los valores observados
print(resX2$observed)
## Grado1 Grado2 Grado3 Grado4 Grado5 
##   2900    392    932   1812   2854
# Ver los resultados de la prueba
print(resX2)
## 
##  Chi-squared test for given probabilities
## 
## data:  inscritos
## X-squared = 2842.8, df = 4, p-value < 2.2e-16

Las hipÃ³tesis a contrastar son:

  • H0: no existe diferencia entre los recuentos de las personas inscritos en los grados.

  • Ha: existe diferencia entre los recuentos de las personas inscritos en los grados.

El p-valor < 2.2e-16 es menor a el p-valor crÃ­tico de 0.05, por lo que se rechaza H0, y decimos que existe diferencia entre los recuentos de las personas inscritas en los diferentes grados evaluados.

Chi cuadrado de independencia de dos factores

Es el uso mÃ¡s tÃ­pico de la prueba es el contrastar si los niveles de dos factores, es decir, de dos variables categÃ³ricas, son independientes o existe evidencia estadÃ­stica de asociaciÃ³n entre ellas.

Las hipÃ³tesis a contrastar son:

  • H0: las variables son independientes.

  • Ha: las variables son dependientes.

Veamos si existen diferencias entre los factores en los dos ejemplos cargados

##

print(class(grados))
## [1] "matrix" "array"

El p-valor < 2.2e-16 es menor a el p-valor crÃ­tico de 0.05, por lo que se rechaza H0, y decimos que existe dependencia entre las variables niveles y grados.
